# Task 15: Mini Single-Page Application (SPA) Using Vanilla JavaScript

## Overview

This task demonstrates building a simple Single-Page Application using Vanilla JavaScript without frameworks.

## Features

- Client-side routing without page reloads
- Dynamic view rendering
- History API navigation (pushState, popstate)
- Loading indicator between views
- 404 fallback handling

## Technologies Used

HTML5, CSS3, JavaScript (Vanilla, History API)
