# Task 13: Dark & Light Theme Toggle

## Overview

This task demonstrates implementing a persistent dark and light theme toggle using CSS variables, JavaScript, and LocalStorage.

## Features

- CSS custom properties for theme colors
- Toggle button to switch themes
- Theme preference saved in LocalStorage
- Saved theme applied on page reload
- Accessible color contrast

## Technologies Used

HTML5, CSS3 (Variables), JavaScript (LocalStorage)
