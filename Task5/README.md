# Task 5: Contact Form with HTML Validation

## Overview

This task demonstrates creating a contact form using HTML5 form elements and built-in validation attributes.

## Features

- Input fields: name, email, phone, message
- Proper input types (text, email, tel, textarea)
- HTML5 validation using required, maxlength, and pattern
- Labels linked to inputs for accessibility
- Styled form using external CSS

## Technologies Used

HTML5, CSS3
